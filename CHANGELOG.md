# 更新日志

## v0.1 (2026-06-05)

### ✨ 新功能
- 4×2 桌面小组件，实时显示设备信号强度、网络类型、流量、温度、电量
- 主界面仪表盘：一键查看 CPU、内存、存储、温度、网络详情
- 5 套预设主题 + 自定义颜色，支持深色模式
- 应用内更新检查（SHA256 校验、镜像源切换）
- 调试日志查看与导出

### 🏗️ 架构
- CI/CD 4 Job 并行流水线（Preflight → Debug + Release → Publish）
- 自动生成 version.json 与 CHANGELOG.md
- Gradle 构建缓存与并行编译加速
- 自动协议探测（HTTP/HTTPS 智能检测）

### 🐛 修复
- AT+CESQ 信号回退解析（LTE/NR 双分支公式）
- 小组件错误状态全屏覆盖显示
- 镜像源持久化（`.commit()` 立即落盘）
